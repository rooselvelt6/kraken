## Becoming a maintainer

1. Demonstrate sustained contributions over 3+ months
2. Be nominated by an existing maintainer
3. Receive consensus from the current maintainer team

## Responsibilities

- Review and merge PRs
- Triage and respond to issues
- Maintain CI/CD and release process
- Enforce the Code of Conduct
- Guide the project roadmap
